isinstance(<obj>, <type>) - проверка принадлежности объекта <obj> к типу <type>
getattr(<obj>, '<attr>', <default>) - возвращает запрошенный атрибут <attr> или значение по умолчанию <default> у <obj>
setattr(<obj>, <name>, <value>) - создаёт у <obj> атрибут с именем <name> и значением <value>
delattr(<obj>, <value>) - удаление атрибута у объекта если это возможно
hasattr(<obj>, <attr>) -> True | False - проверяет на наличие атрибута <attr> в объекте <obj>
Атрибуты класса - переменные хранящиеся внутри
Методы класса - функции находящиеся внутри
.__doc__ - документация класса
.__dict__ - атрибуты класса

__init__ - магический метод инициализации. Отвечает за процесс установки начального состояния

Паттерн моносостояние - __shared_attr общий словарь. Необходим для установки и изменения, одних и тех-же, значений у ЭК
Защищённость атрибутов класса:
.atr - публичный <public> доступ к атрибуту
._atr - защищённый <protected> доступ к атрибуту
.__atr - закрытый <private> доступ к атрибуту

getter-метод и setter-метод
getter - метод используется для получения атрибута ЭК
setter - метод для установки значения атрибута ЭК
property(fget=, fset=, fdel=) - функция/декоратор позволяющая определить метод класса как атрибут

@property - один из вариантов использования: предоставить атрибуты только для чтения в ваших классах
